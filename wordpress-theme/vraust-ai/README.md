# Vraust.ai WordPress Theme

A premium, enterprise-grade WordPress theme for Vraust.ai - Privacy-preserving AI infrastructure for fraud and financial crime detection.

## Installation

1. Download the `vraust-ai` folder
2. Upload to `/wp-content/themes/` on your WordPress installation
3. Go to **Appearance > Themes** in WordPress admin
4. Activate "Vraust.ai"

## Setup

### Create Demo Page
1. Go to **Pages > Add New**
2. Title it "Demo" (the slug should be `demo`)
3. In Page Attributes, select "Demo Request" template
4. Publish

### Configure Navigation
1. Go to **Appearance > Menus**
2. Create a new menu and assign to "Primary Menu"
3. Add links: How It Works (#how-it-works), Use Cases (#use-cases), About (#about)

### Customize Content
All content is customizable via **Appearance > Customize**:
- **Hero Section**: Headlines, badges, CTAs
- **Problem Section**: Cards and descriptions
- **Solution Section**: Pillars and vault content
- **How It Works**: Steps and diagram message
- **Use Cases**: All 4 use cases
- **Target Clients**: Client types and logos
- **Trust Section**: Credentials and quote
- **CTA Section**: Final call-to-action
- **Demo Page**: Form text and email recipient
- **Footer**: Description and copyright
